2. TITLE NORMALIZATION

When scraping job titles:

Convert to lowercase and remove punctuation.

Example:

"Senior Software Engineer" → "software engineer"
"AI/ML Engineer" → "ai engineer"

---

3. TITLE MAPPING

Match scraped titles to the closest predefined role.

Example mappings:

Senior Software Engineer → Software Engineer
Junior Software Developer → Software Engineer
React Developer → Frontend Developer
Node.js Developer → Backend Developer
ML Engineer → Machine Learning Engineer
AI/ML Engineer → AI Engineer
DevOps Specialist → DevOps Engineer

---

4. TITLE CLEANING

Before saving:

Trim whitespace
Remove newline characters
Remove prefixes such as:

Senior
Junior
Lead
Principal

Keep the standardized role title.

---

5. SAVE STANDARDIZED TITLE

Save only the normalized role in the database.

Example:

Original scraped title:

"Senior AI/ML Engineer"

Saved title:

"AI Engineer"

6. FILTER RESULTS

Ensure jobs shown in dashboard belong to one of the predefined roles.
